# Jihun (Max) Joun - Academic Website

Personal academic website for Dr. Jihun (Max) Joun, Assistant Professor of Digital Forensics at Arizona State University.

## About

This website showcases my research, publications, teaching, and professional activities in the field of digital forensics and cybersecurity.

## Website URL

https://jihunjoun.github.io

## Features

- Responsive design that works on desktop, tablet, and mobile devices
- Clean, professional layout inspired by modern academic websites
- Sections for:
  - About & Research Interests
  - News & Updates
  - Publications
  - Research Projects
  - Software & Tools
  - Teaching Experience
  - Professional Service

## Setup Instructions

1. Create a new repository on GitHub named `jihunjoun.github.io`
2. Upload the following files to the repository:
   - `index.html`
   - `style.css`
   - `README.md`
3. Go to repository Settings → Pages
4. Set Source to "Deploy from a branch"
5. Select the `main` branch and `/ (root)` folder
6. Click Save
7. Your website will be live at https://jihunjoun.github.io within a few minutes

## Customization

### Updating Content

Edit `index.html` to update:
- Personal information
- Publications
- Research projects
- Teaching history
- Service activities

### Changing Colors

Edit `style.css` to modify the color scheme. Current theme uses:
- Primary: #667eea (purple-blue)
- Secondary: #764ba2 (purple)
- Dark: #2c3e50
- Background: #f8f9fa

### Adding Photos

To add a profile photo:
1. Add your photo file to the repository (e.g., `profile.jpg`)
2. In `index.html`, add an `<img>` tag in the `.profile-section` div:
```html
<img src="profile.jpg" alt="Jihun Joun" style="width: 200px; border-radius: 50%; margin-bottom: 20px;">
```
